# Xinyi Xu's Academic Website

This repository hosts the personal academic website for Xinyi Xu, a Ph.D. candidate at the University of Electronic Science and Technology of China (UESTC).

## Overview

This website serves as a comprehensive online profile, showcasing Xinyi Xu's academic journey, research contributions, and contact information. It is designed to be easily accessible and informative for colleagues, collaborators, and prospective students.

## Features

*   **Responsive Design**: Optimized for viewing on various devices, from desktops to mobile phones.
*   **Dynamic Content**: Sections for Education, Publications, Research Interests, and Recent Updates.
*   **Easy Deployment**: Hosted on GitHub Pages for straightforward deployment and updates.
*   **Clean and Professional Aesthetic**: Inspired by modern academic website designs.

## Content Highlights

*   **Education**: Detailed overview of academic degrees, institutions, GPAs, and honors.
*   **Publications**: A list of research papers, including journal articles and conference proceedings, with links to relevant venues.
*   **Research Interests**: Key areas of focus, such as Federated Learning, Agentic AI, Signal Processing, and 6G Networks.
*   **Contact Information**: Easy access to email and social academic profiles.

## Technologies Used

*   **HTML5**: For structuring the web content.
*   **CSS3**: For styling and responsive design.

## Setup and Deployment

This website is designed for deployment on GitHub Pages. To set up your own version or contribute:

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/Lumen-xy/xinyixu.github.io.git
    cd xinyixu.github.io
    ```
2.  **Customize Content**: Edit the `index.html`, `publications.html`, and `education.html` files to update your personal information, publications, and educational background.
3.  **Update Styles**: Modify `css/style.css` to change the website's appearance.
4.  **Replace Profile Picture**: Place your profile image in the `images/` directory, ensuring it is named `profile.jpg`.
5.  **Commit and Push**: Push your changes to the `main` branch of your GitHub repository.
    ```bash
    git add .
    git commit -m "Update website content"
    git push origin main
    ```
